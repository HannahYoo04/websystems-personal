Internal Structure

Common Page Structure

All pages in my website follow a similar internal structure. At the top of each page, there is a header created by the main heading (`h1`), which clearly shows the page title. Directly below the header, I placed the same navigation menu so that users can move between pages in a consistent way. Under the menu, the main content section is arranged using paragraphs, headings, lists, tables, images, and div containers depending on the purpose of the page. For example, some pages use div containers to group related content, while the past page uses a table to organise school information clearly. At the bottom of every page, there is a footer that provides a consistent ending to the layout. This repeated use of a header, menu, content section, and footer helps make the website easier to understand and gives all pages a unified structure.

Graphic Design

Aesthetics

The overall graphic design of the website focuses on a soft, personal, and consistent appearance. I wanted the website to feel warm and friendly rather than formal or complicated. Therefore, I used a simple multi-page layout with the same heading, navigation menu, and footer structure across all pages and I avoided adding too many decorative effects to be clear.

Colour

I mainly used pink and brown tones throughout the website to create a warm and visually consistent theme. Pink is my favourite colour, so I wanted it to be one of the main colours in the design and to reflect my personal taste. The lighter pink backgrounds help make the pages feel soft and personal, while the darker brown colour is used for headings, borders, and important text to create contrast.

Font / Typography

The main page titles use a bold display-style font so that they stand out clearly and immediately show the main topic of each page. For the body text, I used a more traditional serif font to keep the paragraphs readable. I also used bold text in selected places, such as with the span element, to highlight important words without changing the style of the whole paragraph.

Layout/Interactivity

I tried to keep the layout simple and structured by placing the main heading at the top, followed by a centred navigation menu, the main content sections, and a footer at the bottom of each page. I used the same top-down layout on all pages to make navigation predictable and to keep the site visually consistent. On the home page, I used a flexbox layout to place the travel list and image side by side so that the section feels more balanced visually. I also used hover effects on the navigation links to make the menu feel more interactive. In addition, I included background music to make the website feel more personal, and I fixed the audio player to the bottom-right corner so that it remains visible and accessible while the user moves through the site.

Alt Text

I used clear alt text for the images in my website to make their meaning understandable even without the visuals. For example, on the home page, my travel photo uses the alt text “A traveling picture.” On the past page, the logos in my school table use alt text such as “WonderLand Kindergarten logo,” “Seorae elementary school logo,” “Sehwa middle school logo,” “Sehwa high school logo,” and “UTS logo.” By using these descriptions, I made sure that the images still provide information even if they cannot be seen.

Accessibility

My website has been designed to be accessible to as many users as possible. All images include alt text, such as "A traveling picture" and "UTS logo", so that users who cannot see images can still understand the content.

I used clear colour contrast between text and background, such as brown text on a light pink background, to make the content easy to read. The headings are large and clearly separated from the main body text, which helps users understand the structure of each page more easily. The navigation menu is simple, so users can easily understand the structure of the website.

I also considered readability by using body text that is large enough to read comfortably and by adding spacing and padding between sections. This helps users with visual difficulties and users on smaller screens The layout uses standard HTML elements such as headings, paragraphs, lists, and tables, so the content remains understandable.

How the Site Works Without CSS

When CSS is turned off, my website loses its visual design, but it still remains usable because the content is structured with standard HTML elements. The navigation menu still works, the headings and paragraphs remain in order, and the lists and tables still show information clearly. The alt text for images also helps users understand image content even without the visual design.
